# Portfólio - Nikolas Oliveira

Este é um site de portfólio pessoal desenvolvido com HTML, CSS e JavaScript.

## Estrutura do Projeto

- `index.html`: Página principal do site
- `css/style.css`: Estilos CSS para layout e design
- `js/script.js`: Scripts JavaScript para interatividade
- `assets/`: Pasta para imagens e outros recursos

## Como Usar

1. Abra o arquivo `index.html` em um navegador web.
2. Navegue pelas seções: Sobre, Projetos e Contato.
3. O formulário de contato é funcional (simulado).

## Tecnologias Utilizadas

- HTML5
- CSS3 (com Flexbox e Grid)
- JavaScript (Vanilla)

## Personalização

- Edite o conteúdo em `index.html` para personalizar as informações.
- Modifique `css/style.css` para alterar o design.
- Adicione funcionalidades em `js/script.js`.
